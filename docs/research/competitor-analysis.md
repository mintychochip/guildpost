# Competitor Analysis — PvP Server Directory (2026)

## Minecraft Giants

### Minecraft-Server-List.com
- One of the oldest, highest domain authority
- "Web 2.0" feel — dated interface
- Deep category system

### MinecraftServers.org
- Massive traffic
- Sponsored slots cost thousands — barrier to entry
- Broad coverage, not specialist

### Minecraft.buzz
- **Closest to what we're building**
- Modern UI, clean verification process
- Good balance of data + aesthetics
- **Primary design reference**

### TopG.org
- Multi-game giant (Minecraft to Runescape)
- Broad but not expert in PvP niche

## Multi-Game / Rust

### Rustafied
- Community + news hub + directory
- Trusted standard for wipe-cycle tracking

### TrackyServer
- Multi-game: Rust, ARK, CS2
- Data-heavy approach
- **Primary technical competitor for "Watcher" concept**

### BattleMetrics
- **The "final boss"** of server tracking
- Advanced RCON, player tracking
- Our opportunity: provide 20% of their data for free

## New School (2026)

### MinecraftServerList.co
- Rising star
- Java/Bedrock cross-play focus
- High-fidelity video previews

### Admin Coalition / Salty Zombies
- Server networks that host their own directories
- **Hybrid model** — servers + community
